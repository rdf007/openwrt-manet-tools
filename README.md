# OpenWrt MANET Tools

Tools for RF telemetry, mesh observability, and range testing on OpenWrt-based MANET nodes.

## Features
- 802.11s telemetry dashboard
- CSV logging
- Python graph generation
- SNR / MCS / Airtime analysis
- ATAK-oriented field testing

## Structure
- openwrt/ -> CGI and OpenWrt scripts
- python/ -> analysis and plotting tools
- data/ -> sample datasets
