# Roadmap

This document describes potential future directions for the OpenWrt MANET Tools project.

The current implementation already provides:

- lightweight mesh telemetry
- CSV logging
- RF observability
- Python analysis
- browser-based range testing

The roadmap focuses on expanding these capabilities while preserving:

- simplicity
- portability
- low overhead
- field usability
