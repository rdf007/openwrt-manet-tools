# Range Test Protocol

This document describes the recommended methodology for evaluating 802.11s mesh performance using the OpenWrt MANET Tools telemetry dashboard.

The protocol is designed to characterize:

- RF link quality
- PHY adaptation behavior
- mesh stability
- operational range
- real-world usability

under realistic field conditions.

---

# Objectives

The primary objectives of the range test are:

- determine practical communication range
- evaluate mesh stability under movement
- characterize PHY adaptation over distance
- identify degradation thresholds
- evaluate ATAK usability over the mesh
- correlate RF metrics with operational performance
